## Pixy

---

"Pixy" este un joc 2D de tip platformer cu o durata scurta de joc (aproximativ 1–3 minute per sesiune).  
Scopul principal nu este doar finalizarea nivelului, ci si obtinerea unor achievement-uri variate, care incurajeaza rejucarea si explorarea diferitelor moduri de abordare a jocului.

## How to play

---

In acest moment nu exista o versiune incipienta, de test, a jocului.

1. Deschide **Unity**  
2. Selecteaza **Add project from disk** si indica folderul proiectului  
3. Jucatorul si harta vor aparea pe ecran  

**Note:** exista un bug care nu lasa jucatorul sa se miste in anumite generari ale hartii. Lucram sa il rezolvam.
